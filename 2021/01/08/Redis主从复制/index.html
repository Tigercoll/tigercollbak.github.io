问题:如果master 挂了怎么办?

redis-rentine哨兵

哨兵就是来帮我我们把手动需要干的事情,自动化.

安装哨兵

yum install redis-sentinel

新建哨兵配置文件

vim redis_sentin.conf

启动哨兵

[root@10-7-189-100 ~]# redis-sentinel redis_sentinel.conf

可以自己去启几个redis服务模拟一下 master挂了之后 哨兵是如何切换master的.
